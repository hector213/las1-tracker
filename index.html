Dashboard

Associate cards show only non-leadership staff. Leadership appears in the "Meet Your Leadership Team" section above. Cards are sorted alphabetically. Use the search bar to find by name or employee ID, or filter by shift, status, flag, or certification.

The stats bar shows: Active (all active roster), Associates (excluding leadership), In Building (active minus today's call-outs and early-outs, including leadership), and Call Outs / Early Outs for today.

Card Banners

Associate cards display a colored banner when a status is active today: OUT (call-out logged), Early Out (early-out logged), Tardy (late arrival), or a time-off type (Vacation, FMLA, etc.). Banners reset automatically each day. An orange pulsing dot appears when a profile has new activity you haven't viewed.

Associate Profiles

Click any card to open a profile. From there you can add manager notes, toggle certifications, nominate badges, set a Reports To supervisor, and view the private watch notes section (supervisors/admins only). Use the Export PDF button to generate a printable summary for HR records.

Note Types

Coaching: performance conversations, no point impact. Recognition: positive callouts, +10 pts. Leadership: leadership potential observations, +15 pts. Warning: formal warnings, -10 pts. PIP: performance improvement plan entries, -150 pts. General: misc notes, no point impact.

Achievement Badges

Leads, Supervisors, and Admins can nominate associates for badges. Each nomination requires a reason. Nominations show as Pending until an Admin approves them in the Admin panel. Once approved, points are awarded automatically. Badges stack; an associate can earn the same badge multiple times. Hover over a badge to see the nomination reason.

Certifications

Leads, Supervisors, and Admins can toggle certifications on each profile. Each cert earned adds +10 points; removing one deducts 10. There are 26 certifications available covering all major warehouse roles. Use the cert filter on the dashboard to see who is qualified for a specific role.

Private Watch Notes

The amber dashed section at the bottom of each profile is only visible to Supervisors and Admins. Use it to track attendance patterns, early departures, tardiness, or behavioral observations you don't want in the general notes feed. These notes do not affect points.

Shift Snapshots

Leads, Supervisors, and Admins can import productivity snapshots via the Import Snapshot button on the dashboard. Three methods are available:

  • Upload Image: Take a screenshot of the shift report — OCR will extract the data automatically.
  • Paste Snapshot: Select all and copy the shift report, then paste it into the text box.
  • Manual Entry: Type in the Employee ID, date, UPH, and other fields directly.

Snapshots appear on the associate's profile under the Shift Snapshots tab, showing UPH, units, target, and on-pace status over time.

Team Attendance Tracker

The Attendance sidebar tab shows a team-wide view of all logged attendance events. Features include:

  • At-Risk Panel: Shows up to 8 associates with the most points in the selected period, with policy step labels.
  • Period Filter: Scope the view to This Week, This Month, This Year, or All Time.
  • Log Table: Filter by associate, shift, event type, and date range. Associate names are color-coded by policy status.
  • Calendar: Visual month calendar showing which days had events.
System Credentials

The System Credentials panel on each profile stores system logins (WMS, scheduler, etc.) tied to that associate. Leads and above can add entries; only Admins can delete them. Passwords are masked by default — click the eye icon to reveal or the copy icon to copy without displaying. Credentials are stored in Firebase and are not visible to Viewer-role accounts.

Time Off

The Time Off sidebar tab lets supervisors and admins schedule and track associate time away. Features include:

  • Calendar View: Click any month tab to switch months. Click a day on the calendar to add a time off entry pre-filled with that date. Associates with active entries appear as colored chips on the calendar.
  • Types: Vacation, Unpaid Day Off, FMLA / Medical, Leave of Absence, Pre-Hire Approved.
  • Card Banners: When an entry is active today, a matching banner appears on the associate's dashboard card automatically.
  • Upcoming Events: Time off starting within the next 40 days also appears in the Upcoming Events panel on the dashboard.

Entries are stored per-associate in Firebase and update in real time.

Corrective Action Form

When an associate reaches a warning threshold on the Attendance tab, a Print Corrective Action button appears on their warning tier card. Clicking it generates a print-ready corrective action form that includes:

  • Associate name, employee ID, hire date, and shift schedule
  • Warning tier (Documented Verbal, Written Warning, Final Warning, Termination)
  • Pre-filled attendance summary with all logged events
  • Manager name (Hector Marron) and signature lines
  • Previous discipline history pulled from logged warnings
